# SF Demo Scout

A Claude Code pipeline that configures Salesforce demo orgs based on 
discovery notes and guided sparring.

## Setup

Complete the [Embark + AWS Bedrock setup guide] first, then run:

curl -o ~/.claude/commands/setup-demo-scout.md \
  https://raw.githubusercontent.com/seb-schi/sf-demo-scout/main/setup-demo-scout.md

Then: `claude` → `/setup-demo-scout`
